Quick Start

Get the latest version of Staddle:

bower install staddle

Or

git clone git@github.com:mattbegent/staddle.git

Or download the latest version using the download button at the top of the page.

Install node packages.

cd brilliantwebsitedirectory
npm install

Run Grunt tasks:

grunt

Open http://localhost:8080/ in your browser.

Build a cool website:-).
